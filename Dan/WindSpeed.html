Description:

This program was created to calculate the wind speed of a hurricane using the formula: vm= a (pn-pc) x

Formula Breakdown:

  • Vms is maximum wind speed in knots.
  • a is a constant dependent on location (19-(latitude/5)) (13.6 in Atlantic and E. Pacific, 15 or 14.2 in Indian Ocean).
  • pn is external pressure in mb (ambient/environmental). pc is minimum central pressure in mb (center of eye).


    • The calculation asks the user if they would like the output in km/h or in knots (kn). It also asks the user to specify air pressure measurement type which is either hg or mb. The program then asks the user to enter the external/ ambient pressure, minimum central pressure, and the latitude as a decimal. Finally, it asks the user if they want to calculate another wind speed. If the answer is ‘yes’, it starts over. If the answer is ‘no’ the program closes.
