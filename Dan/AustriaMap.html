Tyrol, Austria Tourism Map

Map Description

This is a tourism map I created of Via-Ferrata locations around Tyrol Austria. The map was meant to give off a feeling of adventure for tourists looking to do via-ferrata’s in the region. The Images cropped into the map are meant to give a sense inspiration and ‘awe’ to encourage tourists to consider visiting some of these beautiful locations.

ArcGIS Desktop was used for preprocessing and creating models for simplifying the open source data that was collected. MAPublisher for Adobe Illustrator was used for visualizing the final cartographic output.

Firstly, all layers were clipped to the Tyrol boundaries. Next roadways and waterways were simplified using tools such as: Dissolve, Trim Line, Smooth Line, and Thin Network. To accurately map the XY points for the Via Ferrata locations an XY event layer was used. I collected X, Y coordinates of each Via Ferrata location along with attribute data and made a CSV file. I then used an XY Event layer tool to import the coordinates as temporary points in ArcGIS Desktop. Finally, I converted the XY points to a feature class so I could display the data as an individual layer in the final cartographic output.
